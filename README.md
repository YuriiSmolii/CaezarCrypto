# Crypto

## Project setup
1. Install python 3.9
2. create venv
```
python3 -m venv env
```
3. activate virtual env
```
source env/bin/activate
```
4. install dependencies
```
pip install -r requirements.txt
```
## Hot to launch

```
python run src/app.py
```

## Known issues
* wxpython does not support python 3.10 at the moment
